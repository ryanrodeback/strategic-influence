## Notation Guide

### Complexity Levels
- **Low:** Minimal rule additions, easy to implement
- **Medium:** Some new systems, moderate implementation effort
- **Medium-High:** Significant changes, careful implementation needed
- **High:** Multiple interacting systems, requires testing
- **Very High:** Complete redesign of core systems, extensive playtesting needed

### Implementation Timeline
- **Easy (1 week):** Simple rule additions
- **Medium (1-2 weeks):** New mechanics affecting several systems
- **Hard (2-3 weeks):** Multiple systems, significant refactoring
- **Very Hard (3-4+ weeks):** Major redesign, extensive playtesting

### Thematic Fit Scale
- **Weak:** Loosely related to inspiration
- **Medium:** Clearly related, some thematic resonance
- **Strong:** Clear thematic implementation
- **Very Strong:** Mechanics embody theme completely
